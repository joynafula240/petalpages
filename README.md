# Petal Pages 🌸

A beautiful, feminine digital book journal designed for readers who want a soft, Pinterest-inspired experience. Track your reading journey with aesthetic cards, gentle animations, and a cozy scrapbook-like feel.

## Features

### 📚 Book Management
- **Add Books**: Easily add new books to your journal with cover images, titles, authors, and personal notes
- **Reading Status**: Track books as "Reading", "Completed", or "DNF" (Did Not Finish)
- **Star Ratings**: Rate your reads from 1-5 stars
- **Personal Notes**: Write reviews and thoughts about each book
- **Favorite Quotes**: Save memorable passages from your reads

### 🌸 Aesthetic Design
- **Soft Color Palette**: Blush pink, cream, lavender, and muted brown tones
- **Feminine Typography**: Elegant serif fonts for headings and handwritten-style fonts for accents
- **Smooth Animations**: Hover effects, transitions, and micro-interactions
- **Responsive Layout**: Works beautifully on both desktop and mobile devices
- **Scrapbook Feel**: Cozy, personal diary-like experience

### 📊 Reading Statistics
- Track total books read
- Monitor currently reading books
- Visual reading goal progress
- Favorite genre insights

### 💾 Data Persistence
- **Local Storage**: All your book data is saved locally in your browser
- **No Login Required**: Start journaling immediately
- **Privacy First**: Your reading journal stays private on your device

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd petal-pages
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Tech Stack

- **React 18**: Modern, component-based UI framework
- **Vite**: Fast build tool and development server
- **Tailwind CSS**: Utility-first CSS framework with custom design system
- **Framer Motion**: Smooth animations and transitions
- **Lucide React**: Beautiful, consistent icon library
- **Google Fonts**: Playfair Display and Caveat for elegant typography

## Design System

### Color Palette
- **Petal Pink**: `#FFE4E6` - Soft background accents
- **Petal Rose**: `#FBCFE8` - Primary button and interactive elements
- **Petal Lavender**: `#E9D5FF` - Secondary accents and status indicators
- **Petal Cream**: `#FEF3C7` - Warm highlights and backgrounds
- **Petal Brown**: `#92400E` - Text and primary content
- **Petal Sage**: `#D1FAE5` - Success states and completed books
- **Petal Mauve**: `#F3E8FF` - Subtle background variations

### Typography
- **Headings**: Playfair Display (elegant serif)
- **Body**: System fonts with serif fallback
- **Accents**: Caveat (handwritten style for personal notes)

### Components
- **Book Cards**: Aesthetic cards with hover effects and status badges
- **Modal System**: Smooth overlays for adding and viewing books
- **Star Rating**: Interactive 5-star rating system
- **Reading Stats**: Animated statistics cards with gradient backgrounds

## Usage

1. **Adding a Book**: Click the "Add New Book" button and fill in the book details
2. **Viewing Details**: Click on any book card to see the full journal entry
3. **Editing**: Use the edit button in the detail view to update book information
4. **Tracking Progress**: Update reading status and dates as you progress through books
5. **Personal Notes**: Add your thoughts, reviews, and favorite quotes to each book
